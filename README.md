Project Description

Knight’s Ascend is a text-based card game where the player ascends a multi-floor tower, battles enemies and strategically manage of cards in turn-based combat. 

Features
Turn-Based Card Combat:
-Player has 10 mana per turn with mana partially regained each turn.
-5 cards available per turn, each card can be used only once per turn.
-Strategic mana management is required to defeat enemies.

Progression System:
-After each battle, players can choose rewards from a set of 3 cards to enhance their deck.
-Permanent health and mana boosts when ascending each floor.

Gameplay Flow:
1. Player enters the tower and goes through the floors.
2. Each floor contains multiple rooms.
3. The last room of a floor contains a boss fight.
4. Combat is conducted using turn-based cards.
5. Player manages mana and card usage strategically.
6. After battles, player can upgrade their deck by selecting reward cards.
7. Ascending to a new floor grants permanent stat boosts.
